Wings-Vaccination-Analysis

You are employed to perform a basic analysis on the data of the country's vaccination drive against covid-19. Use the data that is present in the directory named input and implement the below steps to complete the challenge.

IMPORTANT:

  • Use the respective instructions for Python and Scala for a detailed instructions on how to implement, run and test your code.
  • Getting_started_with_environment_and_validate_cluster_health.pdf
  • Instructions_for_using_Pyspark_with_VSCode.pdf
  • Instructions_for_using_Scala_with_VSCode.pdf

NOTE:

  • Do not make any changes to the exsisting code as it may break the flow of execution
  • Follow the comments in the actual challenge file for easier understanding
  • The extraction and loading methods for the data are already implemented which are extract_data and load_data which has the below functionality.
    • The method extract_data accepts a file as a parameter that is present in the input location and reads the file contents and returns a dataframe.
    • The method load_data accepts a dataframe as a parameter and saves the dataframe into a file in the designated folder.
  • Make sure you use proper naming conventions for the columns names which are marked in a different colour in the table below.
  • There are two seperate folders, one for python and another for scala.
